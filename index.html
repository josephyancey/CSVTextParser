CSVTextParser by josephyancey

A simple tool that will take a CSV and a template and merge them into a single output. It is very handy for generating SQL from CSVs.

Download

You can download this project in either zip or tar formats.

You can also clone the project with Git by running: 

$ git clone git://github.com/josephyancey/CSVTextParser

Template
